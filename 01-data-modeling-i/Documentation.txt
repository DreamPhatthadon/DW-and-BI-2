ได้สร้าง Database โดยใช้ PostgreSQL
โดยตารางแบ่งเป็น 2 ตาราง ชื่อว่า actors, repository และ events

ตาราง actors ใช้เก็บข้อมูลของผู้เขียน (actor) โดยแบ่ง Column สำหรับใช้เก็บข้อมูลดังนี้
id: เก็บข้อมูล id ของ actor แต่ละคน โดยเก็บข้อมูลเป็นจำนวน Integer (int)
login: เก็บข้อมูลชื่อ login ของ actor ว่าใช้ชื่ออะไร โดยเก็บข้อมูลเป็นตัวหนังสือ (text)

ตาราง repository ใช้เก็บข้อมูล repository ของ actor แต่ละคน โดยแบ่ง Column สำหรับใช้เก็บข้อมูลดังนี้
repo_id: เก็บข้อมูล id ของ repository ที่ actor แต่ละคนมี โดยเก็บข้อมูลเป็นจำนวน Integer (int)
repo_name: เก็บข้อมูลชื่อของ repository แต่ละ id โดยเก็บข้อมูลเป็นตัวหนังสือ (text)

ตาราง events ใช้เก็บข้อมูลการกระทำของ actor โดยแบ่ง โดยแบ่ง Column สำหรับใช้เก็บข้อมูลดังนี้
id: เก็บข้อมูล id ของ event แต่ละครั้ง โดยเก็บข้อมูลเป็นตัวหนังสือ (text)
type: เก็บข้อมูลประเภทการทำ event แต่ละครั้ง โดยเก็บข้อมูลเป็นตัวหนังสือ (text)
created_at: เก็บข้อมูลเวลาการทำ event แต่ละครั้ง โดยเก็บข้อมูลเป็นตัวหนังสือ (text)
actor_id int: เก็บข้อมูลว่าการทำ event แต่ละครั้ง ทำโดย actor คนใด โดยเก็บเป็นจำนวน Integer (int)

การนำ Database ไปใช้คือดูว่า events ที่ถูกทำบ่อยที่สุดคืออะไร เวลาไหน ทำโดยใคร ทำผ่าน repository อะไร
เมื่อได้ข้อมูลแล้วสามารถนำไปวิเคราะห์เพื่อพัฒนาธุรกิจต่อไปได้
